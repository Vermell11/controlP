# ControlP Context

ControlP is a local-first visual Agentic Project OS for Andres Ortega. It reads the Cerebro Obsidian vault as the project index, resolves local project folders under `/code`, and visualizes Git, Graphify, sessions, health, and next steps in a V.A.U.L.T.-style dashboard.

Current state: the dashboard listens with precision (V1.4.2, local Whisper STT). Speech recognition runs on-device: `stt/` sidecar (FastAPI + faster-whisper, Spanish, VAD, model via `WHISPER_MODEL`, started with `stt/run.sh`) behind the `/api/stt` proxy — the browser only ever talks to ControlP, and `STT_URL` relocates the sidecar for web deployment without client changes. Audio I/O offers two swappable STT providers (whisper.local default-recommended ↔ web.speech, persisted, spoken fallback hint if the sidecar is down); push-to-talk (hold the 16-band canvas synthesizer or V key). The particle sphere renders voice as a spherical equalizer with damped spectrum attack (no strobing) and smooth state-preset transitions; project orbs glow with voice level. Rule-based command router tolerant to Whisper transcriptions: compact name matching ("MA-I-A" ≡ "MA-IA"), "proyecto" trigger, explicit dictation prefixes (anota/tarea) win over every other rule → intent queue with source voice. All on the V1.3.1 base: three 3D formations, CameraDirector, operational HUD panels, editable records, `/queue`. Data contract `SCHEMA_VERSION 2`.

Quality gates (V1.2.2): `npm run check` (typecheck + explicit ESLint) is the official contract; registry enforces id/slug uniqueness. Note: `next build` hangs in the iCloud folder (`.next` indexing); it builds in ~5 s in a clean local copy — known environmental debt, recommendation is moving the repo out of iCloud. Mic requires secure context (localhost/https).

Reto actual: V1.4.1 — Schedule as operational daily agenda with voice as the capture interface (task with project slug, target date, stable id, open/done status, recorded in project memory on confirm, memory-first transaction), now that voice capture is precise. Afterwards Sprint 4 — LLM assistant (V1.5), which replaces the rule router behind the same `routeCommand` signature. Roadmap in `Proyectos/ControlP/Roadmap`; data architecture in `Proyectos/ControlP/Arquitectura/Datos`.
