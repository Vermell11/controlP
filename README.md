# ControlP

ControlP is a local-first project command OS for the existing traceability system.

V1 is intentionally read-only:

- Obsidian is the project index and human memory.
- Git is implementation evidence.
- Graphify is derived technical context.
- Notion remains the transactional ledger and is not written by this app.
